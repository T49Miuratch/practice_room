# 間違いがちだぞ三浦悠介！
・cssのclassとidを間違いがち！
・class=「.」id＝「#」だぞ！

・<link rel="stylesheet" href="css/style.css">
上記正しい。以下はバツ！　しがち。まちがいがち。
・<style src="css/style.css">

・たぶん以下のscriptと混同してる
一番最初に潰しておきたい部分
<!-- jQueryのライブラリを読み込む -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<!-- JavaScriptのコードを読み込む -->
<script src="js/gsquestjs_rich.js"></script>
